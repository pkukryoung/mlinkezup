Properties Dialog

Purpose

The plan is to confirm the default property pages for simple file, folder, and project resources.

Setup

  • Start the Eclipse workbench
  • Create a simple Project, Folder, and File

Project property pages

  1. Select the simple Project, from the context menu select Properties
  2. Click on the Project References page
  3. Verify that the Project References page includes no project references
  4. Click on the Info page
  5. Verify the content is correct
  6. Check the Read Only option and press OK
  7. Verify that creating another simple file in the simple Project is disallowed

Folder property pages

  1. Select the simple Folder, from the context menu select Properties
  2. Click on the Info page
  3. Verify the content is correct
  4. Check the Read Only option and press OK
  5. Verify that creating another simple file in the simple Folder is disallowed

File property pages

  1. Select the simple File, from the context menu select Properties
  2. Click on the Info page
  3. Verify the content is correct
  4. Check the Read Only option and press OK
  5. Verify that editing the simple File is disallowed (editor is opened but cannot type in it)

Cleanup

  • Delete the simple File, Folder, and Project
