New Folder Wizard

Purpose

The plan is to create a single empty folder. The end result of this wizard should be an empty folder.

Setup

  • Start the Eclipse workbench
  • Create a temporary project called "Folder Test"
  • Select File->New->Folder from the resource perspective

Create folder at project root

  1. Select the project Folder Test
  2. In the Folder Name field, type in f1
  3. Press the Finish button
  4. Verify a new f1 folder is created under the Folder Test project (in navigator view and in the file system)

Cleanup

  • Delete the temporary project "Folder Test"
